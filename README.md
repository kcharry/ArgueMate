#🧠 ArgueMate — Your AI Debate Companion

ArgueMate is an AI-powered platform that transforms how you practice, structure, and improve arguments.

Instead of debating blindly or struggling to structure ideas, ArgueMate guides you through a full debate flow, generates intelligent responses, and helps you sharpen your reasoning in real time.

## 🚀 Key Features
- 🗣️ Structured Debate Flow: Guides users through real debate formats (AC, NC, rebuttals, etc.) Ensures clear, organized argumentation
- 🤖 AI Debate Opponent: Powered by Gemini to generate logical arguments and responses that adapt to your arguments dynamically
- 💡 Customization: Choose difficulty of the debate (Novice, Junior, Open)
- 🔄 Real-time Interaction: Smooth, responsive UI for fast-paced debate practice followed by Immediate AI Judge feedback
- 🎮 Fun learning: Try our mini game, Fact CheckMate to practice identifying strong vs. weak pieces of evidence
- 🧑‍🤝‍🧑 Community Building: Use our search to find schools with Speech and Debate Teams in your area!

## ⚙️ Getting Started
1. Clone the repo
git clone https://github.com/kcharry/ArgueMate.git
2. cd ArgueMate
3. Install dependencies (npm install)
4. Add environment variables.
Create a .env.local file:
GEMINI_API_KEY=your_api_key

5. Run the app
(npm run dev)

Visit: http://localhost:3000
